The problem is not motivation. It is reload friction.

Every switch makes you remember what was open, where the file was, which tabs mattered, and what can safely close. That tiny ritual becomes a tax on every project, every day.

Morning: research spiral 18 tabs, one PDF, zero memory of why half of this exists.
Noon: client mode Wrong files open. Calendar screaming. Desktop doing performance art.
Night: side project The tools are somewhere. The vibe was last seen Tuesday.

Floppy makes context portable.

Instead of rebuilding your setup from memory, you create named disks for the modes you actually live in. Load one, clear one, return later without the scavenger hunt.
